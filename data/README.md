# Data Directory

This directory contains the datasets used in the paired-focus construction
experiments described in the paper.

## Main files

### {cxn_name}.tsv
Primary evaluation dataset for semantics for each construction used throughout the paper. All prediction scripts
expect this format.

### syntactic_tests_combined.tsv
Contains datasets used for syntactic evaluation.

## Archive

The `archive/` directory contains intermediate datasets and earlier versions 
that are not actually used in the paper.
